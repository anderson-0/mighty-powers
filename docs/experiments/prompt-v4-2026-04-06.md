> **4. Security pattern** — copy this exactly into the doc, every API route must follow it:
> ```typescript
> const session = await auth()
> if (!session?.user?.id) return new Response('Unauthorized', { status: 401 })
> const userId = session.user.id
> // every Supabase query: .eq('user_id', userId)
> ```
>
> **5. Library Integration Notes**
> - Auth.js v5: import `auth` from `@/auth`, NOT `next-auth/server`
> - Supabase: service role key bypasses RLS — every query on user data MUST `.eq('user_id', userId)`
> - AI SDK v6: `usage.inputTokens ?? 0`, `usage.outputTokens ?? 0`; `toTextStreamResponse()` not `toDataStreamResponse()`
> - shadcn/ui: Button has NO `asChild` prop — use `<Link className={buttonVariants()}>` for link-buttons; use `AlertDialogTrigger` directly with CSS classes (no nested button)
> - Next.js 16: `proxy.ts` not `middleware.ts`; all request APIs async (`await params`)
> - Model: `'anthropic/claude-haiku-4.5'`
> - GenerateCards fetch: use `response.text()` NOT a streaming reader — required for testability in jsdom

>   process.env.SUPABASE_SERVICE_ROLE_KEY!
> )
> ```
>
> Write `src/lib/usage.ts`:
> ```typescript
> import { supabase } from '@/lib/supabase'
>
> export async function getUsageCount(userId: string): Promise<number> {
>   const startOfMonth = new Date()
>   startOfMonth.setDate(1)
>   startOfMonth.setHours(0, 0, 0, 0)
>   const { count, error } = await supabase
>     .from('generation_logs')
>     .select('*', { count: 'exact', head: true })
>     .eq('user_id', userId)
>     .gte('created_at', startOfMonth.toISOString())
>   if (error) throw error
>   return count ?? 0
> }
>
> export async function logGeneration(userId: string, cardsGenerated: number): Promise<void> {
>   const { error } = await supabase
>     .from('generation_logs')
>     .insert({ user_id: userId, cards_generated: cardsGenerated })
>   if (error) throw error
> }
> ```
>
> Write `proxy.ts` at **project root** (same level as `src/`, NOT inside it):
> ```typescript
> export { auth as default } from '@/auth'
> export const config = { matcher: ['/dashboard/:path*', '/deck/:path*'] }
> ```

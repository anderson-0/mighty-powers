# Deprecated — use `mp:prd`

PRD validation is now the **Validate** intent on **`mp:prd`** (synced from upstream BMAD `bmad-prd`).

**Validate an existing PRD:** invoke `mp:prd` with validate intent (or say "validate my PRD").

The old step files in this directory are retained for reference only — do not follow them when `mp:prd` is available.
